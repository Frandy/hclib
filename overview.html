Habanero-C++ Overview

The different types of supported intra-node asynchronous tasks are as following: 

async([capture_list]() {
  S1;
});

// async with Data-Driven Futures (DDFs)
asyncAwait(ddf_1, ddf_2, ...., [capture_list]() {
  S2;
});

// async with Phasers
asyncPhased(ph_1, mode_1, ph_2, mode_2, ...., [capture_list]() {
  S3;
});

// parallel for loop
forasync(dim, style, ind1, ind2, ..., siz1, siz2, ..., 
			     seq1, seq2, ..., [capture_list]() {
  S4;
});

These asynchronous tasks can be joined by using a finish: 

finish([capture_list]() {
  async(....);
  asyncAwait(...);
  asyncPhased(...);
  forasync(...);
});
